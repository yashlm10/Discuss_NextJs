This is a [Next.js](https://nextjs.org/) project 
## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

---
### Implemented the following
1. Signin/Signout GitHub oAuth authentication in NextJs
2. Used how to configure and use Next UI library.
3. Caching system in NextJs, static and dynamic pages.
4. Client and server components, difference and uses.
5. Forms Validation.
6. Fetching and displaying data from the database.
7. Recursive rendering of comments under posts.
---
### In this web app, user can:
1. Sign in and out of the application, using Github.
2. Create new Topics and view top posts from various topics.
3. View and create posts under a particular topic.
4. View and create a particular post and all the comments (nested) under the post.
